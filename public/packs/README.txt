Pack images — shown on the product page when a customer selects a pack size.

Naming:  {product-slug}-1.jpg  = single bottle (1 pack)
         {product-slug}-2.jpg  = 2 bottles     (2 pack)
         {product-slug}-3.jpg  = 3 bottles     (3 pack)

Right now -1/-2/-3 are all the same single-bottle photo (placeholders).
To show real 2-bottle / 3-bottle shots: just REPLACE the -2.jpg and -3.jpg
files with your own photos (keep the same filename). No code changes needed.
